# Hello World 測試

這是一個簡單的 Hello World 程式。

## 執行方式

### 方法 1：直接執行 JavaScript 檔案
```bash
cd library
node scaffold/hello_world.js
```

### 方法 2：使用相對路徑
```bash
node library/scaffold/hello_world.js
```

## 預期輸出

```
Hello world
```

## 檔案說明

- `hello_world.js` - JavaScript 版本，可以直接用 Node.js 執行
- `hello_world.ts` - TypeScript 版本（需要 ts-node 才能執行）
