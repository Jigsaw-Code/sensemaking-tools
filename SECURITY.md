# Security Policy

## Reporting a Vulnerability

Google Jigsaw takes the security of our open-source projects seriously. If you believe you have found a security vulnerability in `sensemaking-reports`, please do not report it via public GitHub issues.

Instead, please report security issues to the Google Security Team via the Google Vulnerability Disclosure Program (VDP) at:

<https://vulnerability-disclosure-program.external.google.com/>

Please include a detailed description of the vulnerability, steps to reproduce, and any potential impact.
